## Requisitos

Node.js e o NPM instalados.

## Instalando a Aplicação Frontend

```bash
entrar na pasta frontend-cardapio-digital
```

Para instalar as dependências, execute o comando:

```bash
npm install
```

Para executar o projeto basta rodar o comando:

```bash
npm run dev
```
